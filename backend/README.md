# SmartPicks — Backend

Express server (`server.js`) backed by a Python content-based movie recommender
(`recommender.py`).

## Model files

The recommender depends on two pickle files in this directory:

| File | In repo? | Notes |
|------|----------|-------|
| `movies_preprocessed.pkl` | ✅ yes (~2 MB) | Preprocessed movie metadata. |
| `similarity.pkl` | ❌ **not committed** (~176 MB) | Cosine-similarity matrix. Excluded via `.gitignore` because it exceeds GitHub's 100 MB per-file limit. |

`similarity.pkl` is a prebuilt artifact and is **not** generated by any script in
this repo. After cloning, obtain it separately (e.g. shared drive / Git LFS / the
notebook used to build the model) and place it in this `backend/` directory before
running the recommender. Without it, `recommender.py` returns an empty result and
logs `Model files not found`.

## Setup

```bash
npm install                 # Node deps for server.js
pip install -r ../requirements.txt
cp .env.example .env        # then fill in real values
node server.js
```
